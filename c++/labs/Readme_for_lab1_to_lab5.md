# C++ Labs 1 to 5<br>
This repository contains C++ lab exercises from Lab 1 to Lab 5. Each lab demonstrates fundamental C++ concepts such as input/output, operators, control statements, loops, and basic problem-solving logic.<br><br>
## **Lab 1: Basic I/O and Arithmetic Operations**<br>
**Topics Covered:** cout, cin, basic arithmetic, conditional statements, printing patterns.<br>
**Programs:**<br>
1. Print "Hello World".<br>
2. Print Name, Roll No, Department.<br>
3. Print a small poem.<br>
4. Print a motivational quote.<br>
5. Print multiplication table of 5.<br>
6. Print a box using asterisks.<br>
7. Input two numbers and print sum.<br>
8. Input two numbers and print average.<br>
9. Check even or odd number.<br>
10. Calculate area of rectangle.<br><br>

## **Lab 2: Data Types, Operators, and Type Casting**<br>

**Topics Covered:** sizeof(), arithmetic operators, type casting, ASCII values, loops.<br>
**Programs:**<br>

1. Print size of basic data types.<br>
2. Input two numbers and perform arithmetic operations including division/modulus.<br>
3. Convert Celsius to Fahrenheit.<br>
4. Input integer and show floating-point & ASCII equivalent.<br>
5. Evaluate a mathematical equation.<br>
6. Print ASCII values of uppercase letters.<br>
7. Swap two characters using a temporary variable.<br>
8. Find the largest of three numbers.<br>
9. Check if one number is a multiple of another.<br>
10. Print squares and cubes of integers from 0 to 9.<br><br>

10. Halve a number using assignment.<br>
11. Calculate gross salary with HRA and transport allowance.<br>
